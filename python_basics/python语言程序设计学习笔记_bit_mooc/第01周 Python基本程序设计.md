## 程序设计语言概述

也叫编程语言。

分为三种

* 机器语言：01代码
* 汇编语言：增加了人类刻度的助记符
* 高级语言

程序语言已经有600多种，常见的有50多种。

高级语言按照执行方式可以分为两种：

* 编译执行：目标代码执行速度快，称作静态语言，如C和JAVA语言，
* 解释执行：便于维护源代码，称作脚本语言，如Python

## Python语言介绍

2000年推出python 2.0

3.0不兼容2.0.

Python语言是通用语言，是脚本语言，是开源语言，是跨平台语言，是多模型语言（同时支持面向对象和面向过程）。
